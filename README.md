# Priority — Sitio Web

Sitio web de Priority Asesores de Seguros.

## Páginas
- `index.html` — Home
- `salud.html` — Bifurcación de salud (premia vs tradicional)
- `salud-premia.html` — Seguro de salud que te premia (Vitality, con encuesta)
- `salud-tradicional.html` — Seguro tradicional (BMI, Humana, Ecuasanitas, Bupa)

## Deploy
Se sirve como sitio estático con `serve` en Railway (puerto dinámico vía $PORT).
